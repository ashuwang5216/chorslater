# Multilingual Translation Skills

This repository contains three Codex skills focused on improving multilingual and low-resource language translation quality.

## Included Skills

1. `low-resource-translation`
- Improve translation accuracy for low-resource languages and code-switched text.

2. `translation-glossary-enforcer`
- Enforce preferred terminology with a glossary checker script.

3. `multilingual-translation-quality-check`
- Score and audit translation quality before publishing.

## Repository Structure

```text
low-resource-translation/
translation-glossary-enforcer/
multilingual-translation-quality-check/
```

## GitHub Usage

1. Create a GitHub repository.
2. Upload this folder content to the repository root.
3. In Codex marketplace import, use:
- Source: your Git URL
- Git reference: your branch (for example `main`)
- Exact path: `/` if these skill folders are at repo root
